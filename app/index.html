Endpoints

The following is a list of endpoints that are all prefixed with this base URL:

All of these are GET unless otherwise specified.



  • /worlds

    • Returns the entire dataset of worlds

  • /worlds/:worldName

    • Returns information about a specific world

  • /creatures/:creatureName

    • Returns information about a creature: health, type, classification, etc.

  • /npcs/:npcName

    • Returns information about an NPC: locations, quests it gives, etc.

  • /spells/:spellName

    • Returns information about a spell.
    • This does not apply to item cards or treasure cards, they are a separate category.

  • /quests/:questName

    • Returns information about a quest.

  • /mounts/:mountName

    • Returns information about a mount: speed, prices, temporary versions, etc.

  • /jewels/:jewelName

    • Returns information about a jewel.
