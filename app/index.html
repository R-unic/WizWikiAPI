Endpoints

The following is a list of endpoints that are all prefixed with this base URL:




  • GET /worlds

    • Returns the entire dataset of worlds

  • GET /worlds/:worldName

    • Returns information about a specific world

  • GET /creatures/:creatureName

    • Returns information about a creature: health, type, classification, etc.

  • GET /npcs/:npcName

    • Returns information about an NPC: locations, quests it gives, etc.
